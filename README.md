# 📦 Складской учёт — WinForms приложение

Учебный проект: управление остатками товаров с динамическими коллекциями и поиском.

## Функционал

### Добавление товара
- Название (TextBox)
- Количество (NumericUpDown)
- Категория (ComboBox: Электроника, Продукты, Одежда и др.)

### Сортировка и фильтрация
- Кнопка **"Только дефицит"** — показывает товары с остатком < 3 шт.

### Изменение внутри списка
- Кнопки **"+1"** и **"-1"** справа от ListBox
- При нажатии меняется количество выбранного товара напрямую в коллекции `List<StockItem>`

### Живой поиск
- TextBox для ввода текста
- Список фильтруется автоматически (метод `.Contains()`)

### Визуальная индикация
- Если у выбранного товара остаток < 5 шт — красная надпись **"СРОЧНО ТРЕБУЕТСЯ ЗАКУПКА!"**

### Статистика
- Общее количество товаров
- Количество типов товаров
- Количество дефицитных позиций

## Структура классов
- `StockItem` — класс данных (Name, Quantity, Category)
- `MainForm` — вся логика интерфейса и работы с коллекцией

## Как запустить
1. Создать проект Windows Forms в Visual Studio
2. Добавить файлы в проект
3. Запустить (F5)
